개인정보처리방침

시행일: 2026년 3월 1일

1. 개인정보의 수집 및 이용

"읽어줘"는 사용자의 개인정보를 수집하지 않습니다.

본 앱은 회원가입, 로그인 과정이 없으며, 사용자의 이름, 이메일, 전화번호 등 개인을 식별할 수 있는 정보를 수집하거나 저장하지 않습니다.

2. 기기 내 저장 데이터

앱의 정상적인 기능 제공을 위해 아래 데이터가 사용자의 기기 내에만 저장됩니다. 이 데이터는 외부 서버로 전송되지 않습니다.

데이터 항목 용도
독서 위치 및 진행률 이어 읽기 기능
북마크 및 메모 사용자 북마크 관리
독서 통계 일별/주별/월별 독서 시간 표시
앱 설정 테마, 글꼴, 음성 속도 등 환경설정 유지

앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

3. 앱 권한 사용 안내

본 앱은 아래 기기 권한을 사용하며, 명시된 목적 외에는 사용하지 않습니다.

권한 사용 목적
저장소 접근 기기에 저장된 전자책 파일 불러오기
마이크 졸음 감지 기능 (호흡 분석, 선택적)
센서 졸음 감지 기능 (움직임 분석, 선택적)
인터넷 AI 요약 및 설명 기능
알림 백그라운드 음성 재생 시 미디어 컨트롤

4. AI 기능 관련 데이터 처리

책 요약 및 텍스트 설명 기능 사용 시, 해당 텍스트 내용이 AI 서비스(Google Gemini)로 전송됩니다. 이는 요약 및 설명 생성을 위한 것이며, 전송된 데이터는 저장되지 않습니다.

AI 기능은 선택적이며, 사용하지 않을 경우 외부로 데이터가 전송되지 않습니다.

5. 제3자 제공

본 앱은 사용자의 데이터를 제3자에게 판매, 공유, 또는 제공하지 않습니다.

6. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만의 사용자가 개인정보를 제공한 사실을 알게 된 경우, 해당 정보를 즉시 삭제하겠습니다.

7. 개인정보처리방침의 변경

본 방침이 변경될 경우, 앱 내 공지 또는 본 페이지를 통해 사전에 안내드립니다. 변경된 방침은 게시한 날로부터 효력이 발생합니다.
